# YouTube Video Summarizer 

A simple tool that helps to summarize a YouTube video with the help of AI saving time and giving insights.

The system helps the user to understand the video by giving the following output:
1. Short Summary
2. Detailed Summary
3. Key Bullet Points
4. Actionable Insights
5. Tone

---

## Features

- Takes YouTube URL and output language as input  
- Works even without captions using speech-to-text  
- Handles long videos using chunking  
- Supports multilingual summaries  
- Optimized for free-tier API usage  
- Generates clean and structured summaries  

---

## How It Works
1. The tool takes two primary inputs from the user: a YouTube video URL and a preferred output language—either English or the original language of the video.
2. If the transcript is available, the tool summarizes it and gives the output
3. If the transcript is not available, the tool analyzes the video using speech-to-text, summarizes it and gives the output
